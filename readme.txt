Android Studio (Activity Tool) v1.0.2
======================================

Thank you for choosing Android Studio (Activity Tool)!

This application helps keep your Windows system active by simulating user activity (mouse and keyboard), preventing your computer from going idle or locking automatically. It is designed for professionals who need to keep their system awake during long tasks, presentations, or remote work.

Key Features:
-------------
- Simulates mouse and keyboard activity with human-like patterns
- Modern, user-friendly interface
- System tray integration and global hotkey support
- Configuration management and persistence
- Resource usage monitoring (CPU, RAM, GPU)
- Easy uninstallation and reset to defaults
- Professional documentation and support

Installation Notes:
-------------------
- The installer will guide you through the setup process.
- You may choose to create a desktop shortcut during installation.
- The application requires Python 3.13 to be installed on your system.
- Administrator privileges are required to run the application.
